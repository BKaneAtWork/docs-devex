= Search for Documents
:description: The Search Service lets you create a customizable search experience for your database. 
:page-topic-type: concept

[abstract]
{description}

The Search Service uses Search indexes and Search queries to collect and return content from your database. 

== Search Indexes

Create a Search index xref:create-search-index-ui.adoc[with the UI] or xref:create-search-index-rest-api.adoc[from the REST API] to control the content that's available in your search results. 

== Search Queries 

Run a Search query xref:simple-search-ui.adoc[with the UI] or xref:simple-search-rest-api.adoc[from the REST API] to return content from a Search index.
